# KeebsForID

KeebsForID adalah website katalog mechanical keyboard berbasis Laravel yang dibuat untuk memenuhi tugas UTS Startup Teknologi.

## Fitur
- Login dan Register
- Dashboard Admin
- CRUD Keyboard
- Catalog Keyboard
- Catalog Switches
- Catalog Keycaps
- Community Page
- Responsive Design

## Teknologi
- Laravel
- PHP
- MySQL
- HTML
- CSS

## Cara Menjalankan
```bash
composer install
npm install
php artisan migrate
php artisan serve
